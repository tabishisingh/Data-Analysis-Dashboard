# Sales Performance Data Analysis(Interactive Dashboard Creation using MS Excel)
## Project Objective
The objective of this project is to analyze the sales performance of sales executives across different regions and evaluate how effectively targets were achieved. This dashboard helps management identify top performers, regional performance trends, and target achievement gaps, enabling better planning and performance improvement strategies.


## Dataset Used
<a href="https://github.com/tabishisingh/Data-Analysis-Dashboard/blob/main/excel%20project.xlsm">Sales Performance Data </a>


## Questions (KPIs)
- What is the total sales performance of each sales executive?
- Which sales executive performed best based on target hit percentage?
- Which region contributes the highest total sales?
- How many sales executives achieved or missed targets?
- What is the average target achievement percentage?
- Comparison of actual sales vs target sales
- Identify executives who are far away from target

- Dashboard Interaction <a href="https://github.com/tabishisingh/Data-Analysis-Dashboard/blob/main/Screenshot%202026-02-03%20234740.png">View dashbaord</a>

## Process
- Verified the dataset for missing values and inconsistencies.
- Ensured correct data types and standardized formats.
- Calculated Total Sales, Target Hit %, and Away From Target %.
- Created pivot tables to analyze:
- Sales by region
- Sales by executive
- Target achievement analysis
- Designed an interactive dashboard using:
- Pivot charts
- Slicers for Region and Sales Executive
- Combined all visuals into a single, user-friendly dashboard.

## Project Insights
- The dataset contains 141 sales executives across multiple regions.
- The average target achievement is approximately 55%, indicating scope for improvement.
- Nagpur region contributes the highest total sales overall.
- Jagdish Chandra is the top-performing sales executive with the highest target hit percentage.
- Several executives are significantly away from target, highlighting the need for focused coaching and strategy changes.
- Regional performance varies, suggesting region-specific sales strategies can improve results.

## Final Conclusion

To improve overall sales performance, the organization should focus on underperforming regions and sales executives through targeted training and performance monitoring. Best practices from top-performing executives and regions should be identified and replicated. Using this interactive Excel dashboard, management can continuously track progress, optimize sales strategies, and improve target achievement in future sales cycles.
